# 文件移动方法实现计划

## 功能描述
创建一个 `move_files_between_folders` 函数，用于将源文件夹中的所有文件移动到目标文件夹中。

## 实现步骤

### 1. 函数设计
- **函数名**：`move_files_between_folders`
- **参数**：
  - `source_folder`：源文件夹路径
  - `target_folder`：目标文件夹路径
  - `overwrite`：是否覆盖目标文件夹中的现有文件（默认 False）
  - `recursive`：是否递归处理子文件夹（默认 False）
  - `extensions`：指定要移动的文件扩展名列表（默认 None，移动所有文件）

### 2. 核心功能
- **路径检查**：验证源文件夹是否存在且不为空
- **目标文件夹创建**：如果目标文件夹不存在，自动创建
- **文件筛选**：根据指定的扩展名筛选要移动的文件
- **文件移动**：使用 `shutil.move` 移动文件
- **覆盖处理**：根据 `overwrite` 参数决定是否覆盖现有文件
- **递归处理**：根据 `recursive` 参数决定是否处理子文件夹

### 3. 错误处理
- 源文件夹不存在
- 源文件夹为空
- 文件移动失败
- 权限错误

### 4. 日志输出
- 开始移动前的统计信息
- 每个文件的移动状态
- 移动完成后的汇总统计

### 5. 使用示例
提供详细的使用示例，包括：
- 基本用法
- 递归移动
- 指定文件扩展名
- 覆盖现有文件

## 技术实现
- 使用 `pathlib` 进行路径操作
- 使用 `shutil` 进行文件移动
- 使用 `try-except` 进行错误处理
- 使用 `tqdm` 显示移动进度（如果可用）
